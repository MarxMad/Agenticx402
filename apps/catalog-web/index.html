Sustituye /ruta/a/tu/clon por la ruta absoluta de tu repo. Para 402, define STELLAR_SECRET_KEY en el entorno del proceso MCP. 

{
  "mcpServers": {
    "pumax402": {
      "command": "node",
      "args": ["/ruta/a/tu/clon/Agentix402/apps/mcp/server.mjs"],
      "cwd": "/ruta/a/tu/clon/Agentix402"
    }
  }
}

Comprueba el nombre de carpeta (Agentix402 vs Agenticx402).
