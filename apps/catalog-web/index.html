Catálogo Documentación

    Para desarrolladores y agentes

    PumaX402 es un hub de descubrimiento para APIs con pago por petición vía x402 en Stellar. Esta web lista el catálogo; el flujo de pago (402 → firma → reintento) lo hacen el CLI o el MCP con tu wallet testnet.

    Guías en el repositorio

    • README — visión, fases y plan
    • CLI — list, fetch, call, variables de entorno
    • MCP — servidor para Cursor / clientes MCP (list_services, call_service)
    • Setup Fase 0 — wallet testnet y primer 402 real
    • PROGRESS — avance del equipo
    • Hackathon / jurado — qué resolvemos, Stellar y x402, deploy sin Docker, guion del vídeo
    • Panorama x402 en Stellar — servicios que ya existen + ideas nuevas
    • Contribuir

    Cursor — ejemplo MCP (mcp.json)

    Sustituye /ruta/a/tu/clon por la ruta absoluta donde clonaste el repo. Para pagar 402, define STELLAR_SECRET_KEY en el entorno del proceso MCP (no solo en el terminal). 

    {
  "mcpServers": {
    "pumax402": {
      "command": "node",
      "args": ["/ruta/a/tu/clon/Agentix402/apps/mcp/server.mjs"],
      "cwd": "/ruta/a/tu/clon/Agentix402"
    }
  }
}

    Comprueba la ruta exacta de tu carpeta (p. ej. Agentix402 vs Agenticx402).
